[라우터 정리]

1. centerRouter
/ -> 홈
/challenge -> 챌린지 (개별 챌린지는 /challenge/개별id로 표기)
/community -> 커뮤니티
/login -> 로그인
/join -> 회원가입
/faq -> 자주묻는 질문

2. userRouter
/users/개별id -> 마이페이지
/users/edit -> 개인정보수정
/users/logout -> 로그아웃

3. adminRouter
/admin/개별id/mySales -> 관리자페이지(나의 판매)
/admin/개별id/productReg -> 관리자페이지(상품 등록)

4. storeRouter(완료)
/store/best -> /쇼핑/베스트
/store/fashion -> /쇼핑/패션
/store/equipment -> /쇼핑/기구
/store/food -> /쇼핑/식품
/store/개별id/details -> 제품별 상세정보
